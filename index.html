# How to install?

Well, for you to begin enjoying Sass Devtools, you need to download the file .scss and incorporate it in your file .scss.

@import "sass-devtools.scss"

# Mixins

Sass Devtools, has mixins that will help your productivity. As an example mixins of effects Transition, Border-Radius, Box-Shadow, Box-Sizing.




## Mixin of Transition

The mixin transition is very useful for you to apply effects and rapid transition without typing a lot.



@include transition(all, 300ms, ease);
Values Information
All Method of Transition.
300ms Time of Transition.
Ease Type of Transition.





## Mixin of Border Radius

The mixin Border Radius is very useful for you to apply border without typing a lot.



@include border-radius(30px);
Values Information
30px This is the size of Border Radius.





## Mixin of Box Shadow

The mixin Border Shadow is very useful for you to apply box-shadow without typing a lot.



@include box-shadow(2px, 10px, 30px, 40px, #000);
Values Information
2px This is the Horizontal Offset value.
10px This is the Vertical Offset value.
30px This is the Blur Radius value.
40px This is the Spread Radius value.
#000000 This is the Color value.





## Mixin of Box Shadow with RGBA

The mixin Border Shadow is very useful for you to apply box-shadow without typing a lot.



@include box-shadowRGBA(2px, 10px, 30px, 40px, 0,0,0,0);
Values Information
2px This is the Horizontal Offset value.
10px This is the Vertical Offset value.
30px This is the Blur Radius value.
40px This is the Spread Radius value.
0,0,0,0 This is the Color with RGBA value.

# Functions

Sass Devtools also has great features that will help you, such as conversion values (px to em), (px to %) and (px to rem).




## Function converting PX to EM

body {
width: convertoEM(10);
}

Converts the value 10px to 0.625em




## Function converting PX to %

body {
width: convertoPER(200);
}

Converts the value 200px to 2%




## Function converting PX to REM

body {
width: convertoREM(10);
}

Converts the value 10px to 0.625rem
